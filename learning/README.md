# Shared learning records

Use these templates when adding a topic. Copy all three files into `topics/<topic>/learning/` and replace placeholders before the first lesson:

- `profile.md`: learner goals, experience, environment, and constraints.
- `roadmap.md`: small milestones with verification and exit evidence.
- `log.md`: append-only evidence from completed sessions.

Keep topic records beside the topic rather than in this shared directory. That prevents unrelated learning paths from sharing a status table or evidence log.
