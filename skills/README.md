# Workspace skills

Skills are kept in the repository so their learning workflow evolves with the material they support.

- `learning-coach` contains the shared, subject-neutral coaching loop.
- `<topic>-coach` skills add only knowledge and procedures that are unique to one topic. They may reference the topic's roadmap, official documentation, or specialized validation workflow.

Keep subject-specific references out of `learning-coach`; this prevents one topic from adding irrelevant instructions to every learning session.
